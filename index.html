                    Skills
                

                

                    

                        

                            
    
                                
  • 
                                    
    
                                        
    Sass
    
                                        
    
                                        
    Sass, Syntactically Awesome Style Sheets, is one
                                            of the multiple CSS pre-processor. It reduces repetition of CSS and
                                            therefore saves time for the developer. I was exposed to SASS back from OOPG
                                            (Object Oriented Programming), where I was intrigued by how it really
                                            brought out the beauty of web development (frontend), making it enjoyable.
                                            Since then, I am fully dedicated to use SASS in my other projects.
    
                                    
    
                                
    • 
                                
  •  
                                    
    
                                        
    Networking
    
                                        
    
                                        
    Networking has come a long way since its first
                                            usage in the world. I have dedicated my own free time to take up an online
                                            course, called CompTIA - Network+, which serves as the foundation for my
                                            endeavours, and additional things that I learnt during my internship, which
                                            includes but not limited to server management and networking hardwares.
    
                                    
    
                                
    • 
                                
  •  
                                    
    
                                        
    JavaScript
    
                                        
    
                                        
    JS, or Javascript, being a scripting/programming
                                            language that allows you to dynamically implement complex features to
                                            webpages. I initially was not fond of this concept, and preferred to fully
                                            utilize CSS for all my projects. Little did I know that this was a gem of a
                                            language, and its abilities. From then on, I am "addicted" to Javascript,
                                            even this carousel is made possible, thanks to
                                            Javascript!
    
                                    
    
                                
    • 
                                
  •  
                                    
    
                                        
    HTML5
    
                                        
    
                                        
    HTML 5, or Hypertext Markup Language version 5,
                                            is the standard programming language for describing the contents and
                                            appearance of webpages. Being the basics of web development, I was
                                            introduced to this framework very early during my programming journey. Since
                                            then, I am able to proficiently design any webpages in my liking, and will
                                            continue to do so.
    
                                    
    
                                
    • 
                                
  •  
                                    
    
                                        
    Python
    
                                        
    
                                        
    Python, being a high-level programming language
                                            and used by 80% of developers, has its presence across different computing
                                            fields like artificial intelligence and data science. I started using python
                                            since the age of 11, when I was exploring Linux environment, and wrote
                                            simple automation codes for Wi-Fi exploitation. Since then, I have studied
                                            the essentials of the language and am proficient.
                                        
    
                                    
    
                                
    • 
                                
  • 
                                    
    
                                        
    Atlassian
    
                                        
    
                                        
    Atlassian is a product provider, serving as a
                                            platform (Paas), to software (Saas) and Infrastructue (Iaas). I have great
                                            exposure to this ecosystem, through my internship, which worked based on its
                                            applications such as Jira, Confluence, and Bitbucket. I am well-versed with
                                            the products' usage and the concept of DevOps, a widely known combination of
                                            cultural philosophies and practices that increases an organization's ability
                                            to deliver applications and services.
    
                                    
    
                                
    • 
                                
  •  
                                    
    
                                        
    Github
    
                                        
    
                                        
    Github is a very popular open source platform
                                            for projects and tools, ranging from official application releases to
                                            independent developer's project works. Other than being a hosting platform
                                            for version control, Github also serves as a coding collaboration tool. I am
                                            well aware of the CLI git control, and the GUI version panel, and have quite
                                            the experience with it, when I used it across different projects.
